Voice Ed: Inclusive Exams for All

Voice Ed is a voice-enabled online examination platform built to make assessments fully accessible for visually impaired students. The system uses Text-to-Speech (TTS) and Speech-to-Text (STT) integration to enable hands-free exam navigation and response entry, while the accessible UI and screen-reader compatibility ensure independent, barrier-free test taking.

  • Voice-based navigation and question prompts using TTS.
  • Hands-free answer submission using STT with confidence scoring.
  • Exam creation, scheduling, and automated evaluation tools for instructors.
  • Accessible UI patterns following screen-reader and keyboard-first guidelines.
  • Secure session handling and result export for record keeping.

Technologies: Java, HTML/CSS, SQL, TTS/STT APIs (e.g., Google Cloud Speech), responsive accessible frontend patterns.

Role: Designed the accessible UI, integrated speech APIs, implemented exam management backend, and tested accessibility with screen-readers.
